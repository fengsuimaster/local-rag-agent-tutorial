# 定义

## AC/AC 变换技术概述

### 交流-交流变换
&emsp;&emsp;**交流-交流变换**（AC/AC变换）是指将一种形式的交流电（AC）转换为另一种形式的交流电的技术。通过变换电路，可以改变交流电的电压、频率、相位等参数。

### 交流调压
&emsp;&emsp;**交流调压**是指仅改变交流电电压的变换技术。其主要用于交流电动机的调压调速、降压启动，以及交流设备的调温、调光等场合，对节省能源、提高设备工作效率有重要意义。

### 交流变频
&emsp;&emsp;**交流变频**是指改变交流电频率的变换技术。其主要用于交流电动机的变频调速，使交流电动机的调速变得简单，从而拓宽了交流伺服电机的应用领域。

## 交流调压方式

&emsp;&emsp;交流调压主要有三种实现方式：半波通断控制方式、相位控制方式和斩波控制方式。

### 半波通断控制
&emsp;&emsp;**半波通断控制方式**是指在交流电压过零时刻开通或关断交流开关，使负载与交流电源接通几个半波、关断几个半波，通过改变接通与关断半波的比值（通断比）来控制作用于负载上的电压有效值。由于输出功率与通断比成正比，故也称为**交流调功方式**。

### 相位控制
&emsp;&emsp;**相位控制方式**与相控整流电路的移相触发控制相似，分别在交流电压的正、负半周以相同的触发控制角 \(\alpha\) 开通交流开关，保持交流波形的对称，并通过调节 \(\alpha\) 实现交流调压。

### 斩波控制
&emsp;&emsp;**斩波控制方式**利用脉冲宽度调制（PWM）技术，以等间距方式将交流电压斩波成脉冲序列，通过改变每个脉冲的占空比来调节输出电压的大小。

## 交流变频方式

### 交流-交流直接变频
&emsp;&emsp;**交流-交流直接变频方式**是指将输入的交流电压在一段时间内整流为正电压，另一段时间内整流为负电压，如此交替，使负载得到频率变化后的交流输出电压。该方式的输出频率通常只能远低于输入频率。

### 交流-直流-交流间接变频
&emsp;&emsp;**交流-直流-交流间接变频方式**是指首先利用整流技术将交流电变换为直流电，然后再利用逆变技术将直流电变换为另一频率的交流电。该方式变换能力强，既可输出低于输入频率的交流电，也可输出高于输入频率的交流电。

# 方法

## 交流调压电路

### 半波通断控制电路

#### 电路结构与工作原理
&emsp;&emsp;半波通断控制电路以两个晶闸管 \(\mathrm{T}_1, \mathrm{T}_2\) 反并联构成交流开关，利用交流开关的通断控制作用在负载上的电压。设一个功率控制周期包含 \(N\) 个交流半波，若其中 \(n\) 个半波电压作用在负载上，则输出电压有效值 \(V_o\) 与输入电压有效值 \(V_i\) 的关系为  
\[V_o = V_i \sqrt{\frac{n}{N}} \quad (9.2.1)\]

#### 输出有效值与功率计算
&emsp;&emsp;当负载 \(R_{\mathrm{L}}\) 为纯电阻时，负载上的功率 \(P_o\) 为  
\[P_o = \frac{V_o^2}{R_{\mathrm{L}}} = \frac{V_i^2}{R_{\mathrm{L}}} \frac{n}{N} \quad (9.2.2)\]
其中 \(n/N\) 称为**通断比**。

#### 控制电路与特性分析
&emsp;&emsp;实现半波通断控制的触发电路由锯齿波发生器和电压比较器构成。锯齿波周期等于一个功率控制周期，与控制电压比较后产生控制信号。当控制电压高于锯齿波信号时，输出高电平，触发具有零交叉开启功能的交流开关导通。

&emsp;&emsp;**注：** 功率控制周期 \(N\) 值越大，功率调节的精细度越高，但会导致负载长时间断电，引起白炽灯闪烁或电动机抖动。若 \(N\) 过小，则调节等级过低。采用数字触发电路可将导通半波与关断半波均匀分布，以减小抖动。

&emsp;&emsp;**推论：** 半波通断控制方式仅适用于惯性较大的负载（如电加热器），其优点为电路简单、工作可靠、线路干扰小、功率因数高；缺点是输出电压存在抖动，不适用于惯性较小的负载。

### 相位控制单相交流调压电路

#### 电阻性负载

##### 工作情况分析
&emsp;&emsp;电阻性负载相位控制调压电路的原理图与半波通断控制电路相同（见图9.2.1）。在输入电压 \(u_i\) 的正半周，触发电路在触发控制角 \(\alpha\) 处触发晶闸管 \(\mathrm{T}_1\) 导通；在负半周，在 \(\pi + \alpha\) 处触发晶闸管 \(\mathrm{T}_2\) 导通。触发控制角 \(\alpha\) 可在 \(0 \sim \pi\) 范围内移相。

##### 有效值与功率因数计算
&emsp;&emsp;在电阻性负载下，输出电压有效值 \(V_o\) 与输入电压有效值 \(V_i\) 的关系为  
\[V_o = V_i \sqrt{\frac{2(\pi - \alpha) + \sin 2\alpha}{2\pi}} \quad (9.2.3)\]
&emsp;&emsp;负载电流有效值 \(I_o\) 为  
\[I_o = \frac{V_o}{R_{\mathrm{L}}} = \frac{V_i}{R_{\mathrm{L}}} \sqrt{\frac{2(\pi - \alpha) + \sin 2\alpha}{2\pi}} \quad (9.2.4)\]
&emsp;&emsp;交流电路输入功率因数 \(\lambda\) 为  
\[\lambda = \frac{P_i}{S} = \frac{P_o}{S} = \frac{V_o}{V_i} = \sqrt{\frac{2(\pi - \alpha) + \sin 2\alpha}{2\pi}} \quad (9.2.5)\]
&emsp;&emsp;**注：** 式(9.2.5)忽略了电路损耗和谐波影响。由于相控调压电路输出电流滞后于电压，且存在谐波，功率因数较低，尤其当 \(\alpha\) 较大时更低。

##### 谐波分析
&emsp;&emsp;由于输出电压 \(u_o\) 的正、负半波对称，不含直流分量和偶次谐波。其傅里叶级数表示为  
\[u_o = \sum_{n = 1,3,5,\dots}^{\infty}(a_n \cos n\omega t + b_n \sin n\omega t) \quad (9.2.7)\]
&emsp;&emsp;基波电压幅值 \(V_{1m}\) 为  
\[V_{1m} = \sqrt{a_1^2 + b_1^2} \quad (9.2.10)\]
&emsp;&emsp;第 \(n\) 次谐波电压幅值 \(V_{nm}\) 为  
\[V_{nm} = \sqrt{a_n^2 + b_n^2} \quad (n = 3,5,\dots) \quad (9.2.13)\]
&emsp;&emsp;**推论：** 谐波次数越低，幅值越大。3次谐波最大值出现在 \(\alpha = \pi/2\) 处，幅值约为基波的 \(1/3\)；5次谐波最大值出现在 \(\alpha = \pi/3\) 和 \(2\pi/3\) 处，幅值约为基波的 \(1/7\)。

#### 电感性负载

##### 负载阻抗角
&emsp;&emsp;对于电感性负载，负载阻抗角 \(\phi = \arctan(\omega L / R_{\mathrm{L}})\)，相当于电感性负载直接在正弦交流电压作用下，电流滞后于电压的角度。根据触发控制角 \(\alpha\) 与 \(\phi\) 的关系，分为三种情况。

##### \(\alpha > \phi\) 情况
&emsp;&emsp;当 \(\alpha > \phi\) 时，晶闸管导通角 \(\theta < \pi\)。负载电流 \(i_o\) 在 \(\alpha \sim \alpha + \theta\) 区间内的表达式为  
\[i_o = \frac{\sqrt{2} V_i}{X_{\mathrm{L}}} \sin(\omega t - \phi) - \frac{\sqrt{2} V_i}{X_{\mathrm{L}}} \sin(\alpha - \phi) \mathrm{e}^{(\alpha - \omega t)/\tan \phi} \quad (9.2.16)\]
其中 \(X_{\mathrm{L}} = \sqrt{R_{\mathrm{L}}^2 + (\omega L)^2}\)。导通角 \(\theta\) 与 \(\alpha, \phi\) 的关系满足  
\[\sin(\theta + \alpha - \phi) = \sin(\alpha - \phi) \mathrm{e}^{-\theta / \tan \phi} \quad (9.2.17)\]
&emsp;&emsp;输出电压有效值 \(V_o\) 为  
\[V_o = V_i \sqrt{\frac{\theta + \sin 2\alpha - \sin 2(\alpha + \theta)}{\pi}} \quad (9.2.18)\]
&emsp;&emsp;负载电流有效值 \(I_o\) 为  
\[I_o = \frac{\sqrt{2} V_i}{X_{\mathrm{L}}} \sqrt{\frac{\theta}{\pi} - \frac{\sin \theta \cos(2\alpha + \phi + \theta)}{\pi \cos \phi}} \quad (9.2.19)\]

##### \(\alpha = \phi\) 情况
&emsp;&emsp;当 \(\alpha = \phi\) 时，导通角 \(\theta = \pi\)，输出电压和电流连续。由式(9.2.16)可知，第二项为0，负载电流即为输入电压直接作用于负载产生的正弦电流。此时电路失去调压能力。

##### \(\alpha < \phi\) 情况
&emsp;&emsp;当 \(\alpha < \phi\) 时，若采用普通窄脉冲触发，则只有 \(\mathrm{T}_1\) 反复导通而 \(\mathrm{T}_2\) 始终无法导通，导致电流 \(i_o\) 单一方向，形成直流分量。
&emsp;&emsp;**注：** 为避免直流分量，应采用**宽脉冲**或**脉冲列**对晶闸管进行触发。采用宽脉冲触发时，电路经若干周期后达到稳态，实际导通角变为 \(\phi\)，相当于 \(\alpha = \phi\) 的情况，电路失去调压能力。

##### 触发脉冲要求
&emsp;&emsp;**推论：** 对于电感性负载的单相交流相控调压电路，为可靠工作并实现调压，控制角的移相范围应保持在 \(\phi < \alpha < \pi\) 之间，且晶闸管触发脉冲应采用宽脉冲或脉冲列。

### 三相相控交流调压电路

&emsp;&emsp;将三个单相相控调压电路分别接在对称的三相电源上，并使其触发相角互差 \(2\pi/3\) 相位工作，即构成三相相控交流调压电路。根据晶闸管开关及负载的连接方式，主要有三种形式。

#### 带中线的Y型连接
&emsp;&emsp;带中线的Y型连接三相相控交流调压电路相当于三个单相相控调压电路的组合，每个单相电路通过中线自成回路。各相输出电压、电流波形及控制角 \(\alpha\) 的移相范围均与单相电路相同。

&emsp;&emsp;**注：** 各相三次谐波相位相同，在中线上叠加，产生相当大的三次谐波电流，对变压器和电网不利。因此，这种连接方式很少采用。

#### 无中线的Y型连接
&emsp;&emsp;无中线的Y型连接三相相控交流调压电路的三次谐波电压在中点相互抵消，负载及线路中均无三次谐波电流，因此得到广泛应用。

&emsp;&emsp;**触发信号要求：** 触发信号应与交流电压同步。三个正向晶闸管 \(\mathrm{T}_1, \mathrm{T}_3, \mathrm{T}_5\) 的触发信号互差 \(2\pi/3\) 相位；三个反向晶闸管 \(\mathrm{T}_4, \mathrm{T}_6, \mathrm{T}_2\) 的触发信号也互差 \(2\pi/3\) 相位；同一相两个晶闸管的触发信号互差 \(\pi\) 相位。总触发顺序为 \(\mathrm{T}_1, \mathrm{T}_2, \mathrm{T}_3, \mathrm{T}_4, \mathrm{T}_5, \mathrm{T}_6\)，六个触发信号相位依次差 \(\pi/3\)。

&emsp;&emsp;**注：** 在Y型连接中，至少应有两相导通才能构成电流通道。因此，每个晶闸管的触发信号应为间隔 \(\pi/3\) 相位的**双窄脉冲**或宽度大于 \(\pi/3\) 的**宽脉冲**。

##### 电阻性负载工作情况分析（以 \(\alpha = \pi/6\) 为例）
&emsp;&emsp;设三相负载为电阻性，以A相负载上的相电压 \(u_{\mathrm{AN}}\) 为例进行分析。触发控制角 \(\alpha = \pi/6\) 时，电路工作于三个晶闸管导通和两个晶闸管导通交替的状态，每个晶闸管导通角 \(\theta = \pi - \alpha = 5\pi/6\)。

&emsp;&emsp;**推论：** 无中线Y型连接的三相相控交流调压电路产生的谐波与单相调压电路相同，但三次谐波被电路自身抵消，谐波影响比单相电路更低。

#### \(\Delta\) 型连接
&emsp;&emsp;\(\Delta\)型连接三相相控交流调压电路由三个线电压供电的单相相控交流调压电路组合而成。单相相控交流电路的分析结果完全适用，只需将相电压改为线电压，输入线电流为与该线相连的两个负载相电流之和。

&emsp;&emsp;**优点：** 晶闸管串联在三角形内部，流过相电流，在相同线电流下，晶闸管电流容量可降低。三相对称负载中的三次谐波在 \(\Delta\) 型回路中流动，不出现于线电流中，对供电电路影响较小。

&emsp;&emsp;**局限性：** 仅适用于负载是三个分离得开的单元的情况。

### 斩波控制交流调压电路

#### 电路原理与输出波形
&emsp;&emsp;交流斩波调压电路将脉冲宽度调制（PWM）技术用于交流调压，通过改变脉冲占空比 \(D\) 来实现电压调节。输出波形 \(u_o\) 为输入波形 \(u_i\) 与开关函数 \(G(t)\) 的乘积：
\[u_o = G(t) u_i \quad (9.2.20)\]
&emsp;&emsp;输出电压中含有基波和各次谐波，基波频率为输入电压频率 \(\omega\)，谐波频率分布在开关频率 \(\omega_c\) 及其整数倍频 \(n\omega_c\) 两侧 \(\pm \omega\) 处。开关频率 \(\omega_c\) 越高，谐波越容易滤除。

&emsp;&emsp;**注：** 改变占空比 \(D\) 即可改变基波电压幅值，实现交流调压。

#### 全可控交流开关形式
&emsp;&emsp;交流斩波调压电路使用的交流开关一般采用全可控器件（如IGBT）。常用结构有三种形式：
1.  **单个全可控器件加二极管桥：** 控制电路简单，但导通压降较高（两个二极管和一个全可控器件压降之和）。
2.  **两个全可控器件反向并联加串联二极管：** 两个开关可独立控制，但要求严格同步。
3.  **两个全可控器件共极加反向并联二极管：** 栅极控制电路可共地，电路构成和控制更为方便。

#### 电阻性负载工作方式
&emsp;&emsp;当 \(u_i\) 为正半波时：用斩波脉冲控制 \(\mathrm{T}_1\) 交替通断，\(\mathrm{T}_2, \mathrm{T}_3\) 始终导通，\(\mathrm{T}_4\) 始终截止。\(\mathrm{T}_1\) 导通时，\(u_i\) 加至负载；\(\mathrm{T}_1\) 截止时，负载无电流。
&emsp;&emsp;当 \(u_i\) 为负半波时：用斩波脉冲控制 \(\mathrm{T}_2\) 交替通断，\(\mathrm{T}_1, \mathrm{T}_4\) 始终导通，\(\mathrm{T}_3\) 始终截止。

#### 电感性负载工作方式
&emsp;&emsp;电感性负载下，负载电流 \(i_o\) 滞后于电压，需增加续流开关（如图9.2.16中的 \(\mathrm{T}_3, \mathrm{T}_4\)）。

&emsp;&emsp;当 \(u_i\) 为正半波时：\(\mathrm{T}_2\) 始终导通，\(\mathrm{T}_4\) 始终截止。在 \(i_o < 0\) 期间（电感放电），负载电流经 \(\mathrm{T}_2\) 流动，\(\mathrm{T}_1\) 无法导通；在 \(i_o > 0\) 期间，使 \(\mathrm{T}_3\) 导通，以斩波脉冲控制 \(\mathrm{T}_1\) 交替通断。\(\mathrm{T}_1\) 导通时，\(u_i\) 加至负载；\(\mathrm{T}_1\) 截止时，负载电流经 \(\mathrm{T}_3\) 续流。

&emsp;&emsp;当 \(u_i\) 为负半波时：\(\mathrm{T}_1\) 始终导通，\(\mathrm{T}_3\) 始终截止。在 \(i_o > 0\) 期间，\(\mathrm{T}_2\) 无法导通；在 \(i_o < 0\) 期间，使 \(\mathrm{T}_4\) 导通，以斩波脉冲控制 \(\mathrm{T}_2\) 交替通断。\(\mathrm{T}_2\) 导通时，\(u_i\) 加至负载；\(\mathrm{T}_2\) 截止时，负载电流经 \(\mathrm{T}_4\) 续流。

&emsp;&emsp;**优点：** 与相控调压相比，克服了谐波分量大、功率因数低等缺点。斩波频率越高，输出波形畸变越小，但开关损耗相应增加。

## 交流变频电路

### 单相交流-交流直接变频电路

#### 工作原理
&emsp;&emsp;单相交流-交流直接变频电路由两组晶闸管整流电路反并联构成，分别称为**正组整流器（P组）** 和**反组整流器（N组）**。其工作原理是：将输入交流电压的各个半波在一段时间内全部整流为正半波，叠加形成变频输出电压的正半波；在另一段时间内全部整流为负半波，叠加形成变频输出电压的负半波。通过交替切换P组和N组的工作状态，负载上即可得到频率为 \(f_c\) 的交流输出电压。

&emsp;&emsp;**注：** 为了使变频后的交流正、负半波电压中脉动成分较小，要求输入交流电压的半波密度较高，因此通常采用三相交流电作为输入电源。

#### 无环流控制与有环流控制
&emsp;&emsp;若将P组和N组整流器的封锁与启动同时进行，可能出现两组整流器同时导通的现象，在两组整流器之间产生较大的**环流**。此方式称为**有环流控制方式**，需在两组整流器之间设置环流抑制电感 \(L\)。

&emsp;&emsp;若将原来工作的整流器封锁一段时间，待其完全关断后再启动另一组整流器工作，则两组整流器不会同时导通，不产生环流。此方式称为**无环流控制方式**。

&emsp;&emsp;**注：** 无环流控制方式电路控制复杂，需精确掌握关断所需死区时间，且死区时间会导致输出波形畸变。有环流控制方式控制简单，但需增加环流抑制电感，设备成本高，效率略低。目前大多数交流-交流直接变频电路采用**无环流控制方式**。

#### 输出正弦波电压的控制方法
&emsp;&emsp;为使输出波形近似于正弦波，需使触发控制角 \(\alpha\) 按一定规律变化，保证整流出的每个交流半波的平均电压按正弦规律变化。

&emsp;&emsp;设控制角 \(\alpha = 0\) 时一个交流半波的平均电压为 \(V_{do}\)，则 \(\alpha\) 变化时整流电路输出的每个交流半波的平均电压为  
\[V_o = V_{do} \cos \alpha \quad (9.3.1)\]
&emsp;&emsp;设期望变频输出的正弦波电压为  
\[u_o = V_{om} \cos \omega_o t \quad (9.3.2)\]
&emsp;&emsp;由式(9.3.1)与式(9.3.2)相等可得  
\[\cos \alpha = \frac{V_{om}}{V_{do}} \cos \omega_o t = M \cos \omega_o t \quad (9.3.4)\]
其中 \(M\) 称为**输出电压比**，\(0 \leqslant M \leqslant 1\)。因此触发控制角 \(\alpha\) 为  
\[\alpha = \arccos(M \cos \omega_o t) \quad (9.3.5)\]

&emsp;&emsp;**推论：** 改变 \(M\) 可改变输出电压幅值，改变 \(\omega_o\) 可改变输出电压频率。采用三相桥式整流电路时，每个交流半波间隔为 \(\pi/3\)，在输出电压一个周期内包含的交流半波个数越多，波形越接近正弦波。

### 三相交流-交流直接变频电路

#### 公共交流母线进线方式
&emsp;&emsp;公共交流母线进线方式的三相交流-交流直接变频电路由三组彼此独立的、输出电压相位互差 \(2\pi/3\) 的单相交流-交流直接变频电路组成，均接在经变压器输出的交流电源公共母线上。为防止三组变频电路之间互相干扰，交流电动机的三个绕组必须拆开，共引出6根线。此方式主要用于中等容量的交流调速系统。

#### 输出Y型连接方式
&emsp;&emsp;输出Y型连接方式的三相交流-交流直接变频电路也是由三组彼此独立的、输出电压相位互差 \(2\pi/3\) 的单相交流-交流直接变频电路组成。三组电路在电源端通过三个变压器实现隔离，交流电动机的三个绕组可采用Y型连接，只引出三根线。

&emsp;&emsp;**触发要求：** 在Y型连接方式下，三组桥式电路中至少要有两组桥式电路中的不同相的四个晶闸管同时导通才能构成回路。因此，同一组桥式电路内通过双脉冲触发，不同组之间通过足够宽的脉冲（宽度大于 \(\pi/6\)）保证触发脉冲同时有效的时刻。

### 交流-直流-交流间接变频电路

#### 电路结构与特点
&emsp;&emsp;交流-直流-交流间接变频电路先将交流电整流为直流电，再经逆变电路将直流电变换为另一频率的交流电。整流部分可采用功率二极管整流桥或晶闸管整流桥，滤波部分可采用LC滤波或电容滤波，逆变部分可采用电压型逆变电路或电流型逆变电路。

&emsp;&emsp;**注：** 传统间接变频电路（如采用二极管整流桥加电压型逆变电路）结构简单、成本低，但输入电流谐波大、功率因数低，输出波形为矩形波或阶梯波，含有大量谐波。

#### PWM整流与SPWM逆变的应用
&emsp;&emsp;随着PWM整流技术和SPWM逆变技术的日趋成熟，交流-直流-交流间接变频电路广泛使用了这两项技术，使变频器性能显著提升：

&emsp;&emsp;**电压型间接变频电路：** 输入三相交流电压经PWM整流电路整流，由电容 \(C\) 滤波变换为电压稳定的直流，再经SPWM逆变电路变换为频率和电压均可调的三相交流电。该电路结构对称，改变控制方式可实现电能传送方向的逆转。

&emsp;&emsp;**电流型间接变频电路：** 输入三相交流电压经PWM整流电路整流，由电感 \(L\) 滤波变换为电流恒定的直流，再经SPWM逆变电路变换为频率和电压均可调的三相交流电。同样可实现电能传送方向的逆转。

&emsp;&emsp;**优点：** 输入侧功率因数可接近于1，输出侧波形非常接近于正弦波，对负载工作有利，且可实现能量的双向流动。
