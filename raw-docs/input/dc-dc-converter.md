# 定义

## 开关电源技术概述

### 开关稳压电源与线性稳压电源
&emsp;&emsp;**线性稳压电源**通过工频变压器进行交流电压变换，由整流电路整流为直流电压，由滤波电路滤除纹波，最后由工作在线性放大区的功率晶体管通过调整自身压降来稳定输出电压。其主要缺点是工频变压器体积大、重量大，且功率晶体管损耗大，转换效率低（仅 \(35\% \sim 60\%\)）。

&emsp;&emsp;**开关稳压电源**首先将交流电压整流为直流电压，然后通过功率晶体管的开关动作以脉冲方式实现直流电压变换，并通过调整脉冲占空比稳定输出电压，最后经滤波输出平滑直流电压。

### 开关稳压电源的特点
&emsp;&emsp;与线性稳压电源相比，开关稳压电源具有以下特点：
1.  **功耗小，效率高：** 功率晶体管工作在开关状态，功耗小，变换效率可达 \(70\% \sim 95\%\)。
2.  **体积小，重量轻：** 开关频率高（几十千赫以上），可使用小体积高频变压器传递较大功率，省去了较大的散热片。
3.  **稳压范围宽：** 通过调节脉冲占空比稳定输出电压，输入电压波动较大时仍能保持输出稳定。
4.  **可靠性好：** 一般具有自动保护电路，功率器件发热少，有利于延长使用寿命。
5.  **成本低：** 输出功率几十瓦以上时，成本低于线性稳压电源。

&emsp;&emsp;**注：** 开关稳压电源的主要缺点是电路复杂、输出纹波较高、瞬态响应差，且可能对电网产生开关干扰。

### DC/DC 变换器
&emsp;&emsp;若直接以直流电源作为输入电压，则开关稳压电源中的整流电路可省去，由开关工作方式的电压变换电路和滤波电路直接对直流电压进行变换。这种电路称为**直流-直流变换器（DC/DC变换器）**。按照输入与输出之间是否存在电气隔离，可分为**非隔离型 DC/DC 变换电路**和**隔离型 DC/DC 变换电路**两类。

## 非隔离型 DC/DC 变换电路

### 降压型（Buck）电路
&emsp;&emsp;**降压型（Buck）电路**是一种输出电压 \(V_o\) 小于等于输入电压 \(V_i\) 的非隔离型 DC/DC 变换电路。其原理电路如图6.2.1所示，由开关 S、续流二极管 D、滤波电感 L 和滤波电容 C 构成。

### 升压型（Boost）电路
&emsp;&emsp;**升压型（Boost）电路**是一种输出电压 \(V_o\) 高于输入电压 \(V_i\) 的非隔离型 DC/DC 变换电路。其原理电路如图6.2.9所示，电感 L 位于输入端，称为升压电感。

### 升降压型（Buck-Boost）电路
&emsp;&emsp;**升降压型（Buck-Boost）电路**是一种输出电压 \(V_o\) 既可低于输入电压 \(V_i\)，又可高于输入电压 \(V_i\)，且输出电压极性与输入电压极性相反的非隔离型 DC/DC 变换电路。其原理电路如图6.2.15所示。

## 隔离型 DC/DC 变换电路

### 单端变换器与双端变换器
&emsp;&emsp;隔离型 DC/DC 变换电路中，若功率开关仅能控制电流发生单向通断变化，导致隔离变压器磁芯磁通单方向变化，则称为**单端变换器**（如反激式、正激式电路）。若功率开关可使隔离变压器原边绕组中的电流发生正负方向变化，导致磁芯磁通双向变化，则称为**双端变换器**（如推挽式、半桥式、全桥式电路）。

### 磁通复位原则
&emsp;&emsp;在单端变换器中，功率开关导通时磁通增大，关断时磁通减小。必须保证每个周期内磁通的增大量等于磁通减小量，否则磁芯中的磁通量会逐周期累积，最终导致磁芯饱和，损坏功率开关。这一原则称为**磁通复位原则**。双端变换器的磁通复位由电路工作形式自然解决。

## 电感电流工作模式

### 电感电流连续模式 (CCM)
&emsp;&emsp;若电感电流 \(i_L\) 在整个开关周期内始终大于零，则称为**电感电流连续模式**（CCM）。

### 电感电流断续模式 (DCM)
&emsp;&emsp;若电感电流 \(i_L\) 在开关关断期间下降至零，并在下一开关周期开始前保持为零，则称为**电感电流断续模式**（DCM）。

### 电感电流临界连续状态
&emsp;&emsp;当电感电流 \(i_L\) 在开关关断结束时恰好下降至零，称为**电感电流临界连续状态**。此时对应的电感量称为**临界电感**。

# 方法

## 非隔离型 DC/DC 变换电路

### 降压型（Buck）电路

#### 电路拓扑与工作原理
&emsp;&emsp;Buck 电路由输入直流电源 \(V_i\)、全控开关 S、续流二极管 D、滤波电感 L、滤波电容 C 和负载电阻 \(R_L\) 构成。当开关 S 闭合时，\(V_i\) 向负载供电并向电感 L 储能，电流 \(i_L\) 线性上升；当开关 S 断开时，电感 L 释放储能，电流经续流二极管 D 续流，\(i_L\) 线性下降。

#### 电感电流连续模式 (CCM)

##### 电压增益
&emsp;&emsp;若 Buck 电路工作于 CCM 且忽略器件损耗，则输出电压 \(V_o\) 与输入电压 \(V_i\) 的关系为：
\[V_o = D V_i \quad (6.2.4)\]
其中 \(D = t_{on}/T\) 为开关导通占空比，\(0 \le D \le 1\)。电压增益 \(M = V_o / V_i = D\)。

&emsp;&emsp;**推论：** Buck 电路只能用于降压，输出电压与占空比 D 成正比。

##### 电感电流波形与峰值
&emsp;&emsp;在 CCM 下，电感电流 \(i_L\) 从最小值 \(I_{L\min}\) 线性上升至最大值 \(I_{L\max}\)，纹波峰峰值为：
\[\Delta i_L = \frac{V_i - V_o}{L} D T \quad (6.2.2)\]
电感电流最大值为：
\[i_{L\max} = I_o + \frac{1}{2} \Delta i_L = \frac{V_o}{R_L} + \frac{V_o (1-D) T}{2L} \quad (6.2.8)\]

##### 输出纹波电压
&emsp;&emsp;在 CCM 下，输出纹波电压峰峰值 \(\Delta V_o\) 为：
\[\Delta V_o = \frac{V_o (1-D)}{8 L C f^2} \quad (6.2.13)\]

&emsp;&emsp;**推论：** 增加开关频率 f、增大滤波电感 L 或电容 C 均可降低输出纹波电压。

#### 电感电流连续模式的临界条件
&emsp;&emsp;Buck 电路工作于 CCM 的临界条件为 \(i_{L\min} \ge 0\)，即：
\[L \ge \frac{1-D}{2} R_L T \quad (6.2.15)\]

&emsp;&emsp;**注：** 电感量 L 较小、开关频率 f 较低、负载电阻 \(R_L\) 较大时，电路易进入 DCM。

#### 电感电流断续模式 (DCM)

##### 电压增益
&emsp;&emsp;若 Buck 电路工作于 DCM，则输出电压与输入电压的关系为：
\[V_o = \frac{2 V_i}{1 + \sqrt{1 + 8 \tau / D^2}} \quad (6.2.23)\]
其中 \(\tau = L / (R_L T)\)。电压增益为：
\[M = \frac{2}{1 + \sqrt{1 + 8 \tau / D^2}} \quad (6.2.24)\]

&emsp;&emsp;**推论：** DCM 下输出电压与输入电压、占空比、电感量、负载电阻均有关，呈非线性关系。大功率应用时应避免 DCM。

#### 集成开关调节器 LM2596
&emsp;&emsp;LM2596 是一种典型的降压型开关稳压调节器，采用 PWM 方式工作，固定开关频率为 \(150\,\text{kHz}\)。其输出电压由外接电阻分压决定：
\[V_{\text{OUT}} = 1.235\,\text{V} \frac{R_1 + R_2}{R_1} \quad (6.2.31)\]
LM2596 允许最大输入电压 \(40\,\text{V}\)，可驱动 3A 负载，具有过流、过热保护功能。

### 升压型（Boost）电路

#### 电路拓扑与工作原理
&emsp;&emsp;Boost 电路由输入直流电源 \(V_i\)、升压电感 L、全控开关 S、二极管 D、滤波电容 C 和负载电阻 \(R_L\) 构成。当开关 S 闭合时，\(V_i\) 向电感 L 储能，负载由电容 C 供电；当开关 S 断开时，电感 L 释放储能，与 \(V_i\) 叠加后经二极管 D 向负载供电并给电容 C 充电。

#### 电感电流连续模式 (CCM)

##### 电压增益
&emsp;&emsp;若 Boost 电路工作于 CCM 且忽略器件损耗，则输出电压 \(V_o\) 与输入电压 \(V_i\) 的关系为：
\[V_o = \frac{V_i}{1-D} \quad (6.2.34)\]
电压增益 \(M = V_o / V_i = 1 / (1-D)\)。

&emsp;&emsp;**推论：** Boost 电路只能用于升压，输出电压与开关关断占空比 \((1-D)\) 成反比。

##### 电感电流波形与峰值
&emsp;&emsp;在 CCM 下，电感电流 \(i_L\) 的最大值为：
\[i_{L\max} = \frac{I_o}{1-D} + \frac{V_i D T}{2L} \quad (6.2.37)\]

##### 输出纹波电压
&emsp;&emsp;在 CCM 下，输出纹波电压峰峰值 \(\Delta V_o\) 为：
\[\Delta V_o = \frac{I_o D}{C f} \quad (6.2.39)\]

&emsp;&emsp;**推论：** 增加开关频率 f 或增大滤波电容 C 均可降低输出纹波电压。

#### 电感电流连续模式的临界条件
&emsp;&emsp;Boost 电路工作于 CCM 的临界条件为 \(i_{L\min} \ge 0\)，即：
\[L \ge \frac{D(1-D)^2}{2} R_L T \quad (6.2.41)\]

#### 电感电流断续模式 (DCM)

##### 电压增益
&emsp;&emsp;若 Boost 电路工作于 DCM，则输出电压与输入电压的关系为：
\[V_o = V_i \frac{1 + \sqrt{1 + 2 D^2 / \tau}}{2} \quad (6.2.47)\]
其中 \(\tau = L / (R_L T)\)。电压增益为：
\[M = \frac{1 + \sqrt{1 + 2 D^2 / \tau}}{2} \quad (6.2.48)\]

#### 集成开关调节器 LM2585
&emsp;&emsp;LM2585 是一种多用途开关稳压调节器，可用于 Boost 型或隔离型开关电路。固定开关频率为 \(100\,\text{kHz}\)。其输出电压由外接电阻分压决定：
\[V_{\text{OUT}} = 1.23\,\text{V} \frac{R_1 + R_2}{R_1} \quad (6.2.53)\]

### 升降压型（Buck-Boost）电路

#### 电路拓扑与工作原理
&emsp;&emsp;Buck-Boost 电路由输入直流电源 \(V_i\)、全控开关 S、电感 L、二极管 D、滤波电容 C 和负载电阻 \(R_L\) 构成。当开关 S 闭合时，\(V_i\) 向电感 L 储能，负载由电容 C 供电；当开关 S 断开时，电感 L 释放储能，经二极管 D 向负载供电，输出电压极性与输入电压相反。

#### 电感电流连续模式 (CCM)

##### 电压增益
&emsp;&emsp;若 Buck-Boost 电路工作于 CCM 且忽略器件损耗，则输出电压 \(V_o\) 与输入电压 \(V_i\) 的关系为：
\[V_o = V_i \frac{D}{1-D} \quad (6.2.56)\]

&emsp;&emsp;**推论：** 当 \(D = 0.5\) 时，\(V_o = V_i\)；当 \(D < 0.5\) 时，\(V_o < V_i\)（降压）；当 \(D > 0.5\) 时，\(V_o > V_i\)（升压）。输出电压极性与输入电压相反。

## 隔离型 DC/DC 变换电路

&emsp;&emsp;利用高频隔离变压器实现输入输出电气隔离的 DC/DC 变换电路称为**隔离型 DC/DC 变换电路**。根据高频变压器的连接形式，可分为反激式、正激式、推挽式、半桥式和全桥式电路。

### 反激式电路

#### 电路拓扑与工作原理
&emsp;&emsp;反激式电路由输入直流电源 \(V_i\)、全控开关 \(\mathrm{T}_1\)、高频隔离变压器 TB、整流二极管 D、滤波电容 C 和负载电阻 \(R_L\) 构成。其由升降压型电路演变而来，隔离变压器在电路中起储能-放电作用。

#### 电流连续模式 (CCM)

##### 电压增益
&emsp;&emsp;若反激式电路工作于 CCM，则输出电压 \(V_o\) 与输入电压 \(V_i\) 的关系为：
\[V_o = \frac{N_2}{N_1} \frac{D}{1-D} V_i \quad (6.3.10)\]
其中 \(N_1, N_2\) 分别为变压器原、副边绕组匝数。

#### 电流断续模式 (DCM)

##### 电压增益
&emsp;&emsp;若反激式电路工作于 DCM，则输出电压 \(V_o\) 与输入电压 \(V_i\) 的关系为：
\[V_o = \frac{N_2}{N_1} \frac{D}{D'_{\mathrm{off}}} V_i \quad (6.3.25)\]
其中 \(D'_{\mathrm{off}} = t'_{\mathrm{off}} / T\)，\(t'_{\mathrm{off}}\) 为副边绕组电流下降至零的时间。

#### 两种工作模式比较
&emsp;&emsp;**电流断续模式 (DCM)：**
1.  在输出功率相等的条件下，开关电流最大值小于 CCM，对开关电流容量要求较低。
2.  磁通变化量 \(\Delta \Phi = \Phi_{\max}\)，大于 CCM 的磁通变化量，有利于提高能量传输效率。
3.  输出电流变化率较大，输出电压纹波较大。

&emsp;&emsp;**电流连续模式 (CCM)：**
1.  输出电流变化率较低，输出电压纹波较小。
2.  对开关电流容量要求较高。
3.  在反激式开关电源设计中，一般采用 DCM 以提高工作效率，但小功率电源为追求输出电压平稳性也可采用 CCM。

#### 电路特点
1.  **结构简单，成本低**，广泛用于小功率开关电源。
2.  副边绕组在 \(t_{\mathrm{off}}\) 时段呈现电感特性，起到滤波电感作用，输出端只需并接滤波电容即可使输出电压平稳。
3.  **负载不允许开路**，否则输出电压会很高。实用电路中通常接有**假负载**以防开路。
4.  隔离变压器需有较大储能能力，通常需在磁芯中加入**气隙**以提高储能和能量传输能力。
5.  传输功率 \(P = f V_{\mathrm{e}} \int_{B_r}^{B_w} H \mathrm{d}B\)，当输出功率较大时变压器体积过大，因此通常适用于 **200W 以下** 的开关电源。

### 正激式电路

#### 电路拓扑与工作原理
&emsp;&emsp;正激式电路由输入直流电源 \(V_i\)、全控开关 \(\mathrm{T}_1\)、高频隔离变压器 TB、整流二极管 \(D_1\)、续流二极管 \(D_2\)、滤波电感 L、滤波电容 C 和负载电阻 \(R_L\) 构成。正激式电路由降压型电路演变而来，隔离变压器主要起变压作用，能量在开关导通时直接传输至负载。

#### 磁复位方法
&emsp;&emsp;正激式电路必须保证**磁通复位**。最常用的方法是在隔离变压器中增加一个匝数为 \(N_3\) 的**复位绕组** \(W_3\)，在开关关断时将存留的能量反馈回输入电源。

&emsp;&emsp;若复位绕组与原边绕组匝数相等 (\(N_3 = N_1\))，则最大占空比为 \(50\%\)。

#### 电流连续模式 (CCM)

##### 电压增益
&emsp;&emsp;若正激式电路工作于 CCM，则输出电压 \(V_o\) 与输入电压 \(V_i\) 的关系为：
\[V_o = \frac{N_2}{N_1} D V_i \quad (6.3.32)\]

##### 最大占空比限制
&emsp;&emsp;完成磁复位所需时间 \(t_{\mathrm{rst}} = \frac{N_3}{N_1} D T\) 应小于等于 \(t_{\mathrm{off}}\)，由此可得最大占空比为：
\[D_{\max} = \frac{N_1}{N_1 + N_3} \quad (6.3.36)\]

#### 滤波电感 L 的计算
&emsp;&emsp;一般取滤波电感 L 上的电流变化量 \(\Delta i_L\) 为输出电流 \(I_o\) 的 \(30\%\)，则：
\[L = \frac{\frac{N_2}{N_1} V_i - V_o}{0.3 I_o} t_{\mathrm{on}} \quad (6.3.37)\]

#### 电路特点
1.  隔离变压器直接传输能量，磁芯利用率较高，可构成比反激式电路输出功率更大的开关电源。
2.  隔离变压器不需要储存能量，使用无气隙变压器，绕组峰值电流较小，损耗较小，有利于提高效率。
3.  工作于 CCM，输出电压纹波比反激式电路明显降低，滤波电容容量可较小。
4.  电路较反激式复杂，增加了磁复位电路和滤波电感，成本有所增高。
5.  通常用于构成 **750W 以下** 的开关电源。

### 推挽式电路

#### 电路拓扑与工作原理
&emsp;&emsp;推挽式电路由输入直流电源 \(V_i\)、两个全控开关 \(\mathrm{T}_1, \mathrm{T}_2\)、带中心抽头的隔离变压器 TB、两个整流二极管 \(D_1, D_2\)、滤波电感 L、滤波电容 C 和负载电阻 \(R_L\) 构成。\(\mathrm{T}_1\) 和 \(\mathrm{T}_2\) 交替导通，使隔离变压器双向励磁，副边经全波整流输出。

#### 输出电压与占空比关系
&emsp;&emsp;在电流连续模式下，输出电压 \(V_o\) 与输入电压 \(V_i\) 的关系为：
\[V_o = 2 \frac{N_2}{N_1} D V_i = \frac{N_2}{N_1} D' V_i \quad (6.3.51)\]
其中 \(D = t_{\mathrm{on}} / T\) 为单个开关的占空比，\(D' = 2D\) 为两组开关占空比之和。

#### 电路特点
1.  隔离变压器双向励磁，磁芯利用效率较单向励磁电路有显著提高，可用较小体积变压器传输较大功率。
2.  开关占空比 \(D'\) 可接近 \(100\%\)，输出电压纹波较正激式电路明显降低。
3.  功率开关均为低端开关，驱动电路简单。
4.  存在**偏磁问题**：若电路对称性不好（导通时间差异或通态压降不同），会在变压器原边叠加等效直流电压，导致磁芯单向饱和。
5.  功率开关截止时承受的电压为 \(2V_i\)，对器件耐压要求较高。
6.  通常用于 **低输入电压的大、中功率** 开关电源。

### 半桥式电路

#### 电路拓扑与工作原理
&emsp;&emsp;半桥式电路由输入直流电源 \(V_i\)、两个大容量电容 \(C_1, C_2\)、两个全控开关 \(\mathrm{T}_1, \mathrm{T}_2\)、隔离变压器 TB、两个整流二极管 \(D_1, D_2\)、滤波电感 L、滤波电容 C 和负载电阻 \(R_L\) 构成。\(C_1, C_2\) 构成直流电源中点，电压为 \(V_i/2\)。\(\mathrm{T}_1, \mathrm{T}_2\) 交替导通，变压器原边加载电压为 \(V_i/2\)，双向励磁。

#### 输出电压与占空比关系
&emsp;&emsp;在电流连续模式下，输出电压 \(V_o\) 与输入电压 \(V_i\) 的关系为：
\[V_o = \frac{N_2}{N_1} D V_i \quad (6.3.60)\]
其中 \(D\) 为单个开关的占空比。

#### 电路特点
1.  隔离变压器双向励磁，传输效率、输出电压纹波等性能与推挽式电路一致。
2.  变压器原边仅需一个绕组，成本低。
3.  **无偏磁问题**：电容 \(C_1, C_2\) 的隔直作用可自动平衡直流分量，不易发生磁芯饱和。
4.  功率开关截止时承受的电压为 \(V_i\)，对器件耐压要求较推挽式电路低。
5.  在输入电压相同前提下，要达同等输出功率，流过开关的电流需增加一倍。
6.  开关 \(\mathrm{T}_2\) 为低端开关，\(\mathrm{T}_1\) 为高端开关，驱动电路较复杂。
7.  通常用于 **高输入电压的大功率** 开关电源。

### 全桥式电路

#### 电路拓扑与工作原理
&emsp;&emsp;全桥式电路由输入直流电源 \(V_i\)、四个全控开关 \(\mathrm{T}_1 \sim \mathrm{T}_4\)、隔离变压器 TB、两个整流二极管 \(D_1, D_2\)、滤波电感 L、滤波电容 C 和负载电阻 \(R_L\) 构成。\(\mathrm{T}_1, \mathrm{T}_4\) 为一组，\(\mathrm{T}_2, \mathrm{T}_3\) 为一组，交替导通，变压器原边加载电压为 \(V_i\)，双向励磁。

#### 输出电压与占空比关系
&emsp;&emsp;在电流连续模式下，输出电压 \(V_o\) 与输入电压 \(V_i\) 的关系为：
\[V_o = 2 \frac{N_2}{N_1} D V_i = \frac{N_2}{N_1} D' V_i \quad (6.3.68)\]
其中 \(D\) 为单组开关占空比，\(D' = 2D\)。

#### 电路特点
1.  隔离变压器双向励磁，性能与推挽式、半桥式电路一致。
2.  功率开关截止时承受的电压为 \(V_i\)，对耐压要求较低。
3.  在输入电压相同前提下，要达同等输出功率，对开关电流容量的要求较半桥式电路低。
4.  存在**偏磁问题**，可通过在变压器原边回路串入隔直电容 \(C_1\) 解决。
5.  需要四个功率开关，既有高端开关又有低端开关，驱动电路复杂。
6.  需防止桥臂直通，控制电路复杂。
7.  主要用于 **工业级大功率** 开关电源，输出功率可做得最大。

### 各种隔离型 DC/DC 变换电路的特点总结

| 电路形式 | 励磁方式 | 输出功率 | 开关耐压 | 偏磁问题 | 驱动电路 | 典型应用 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **反激式** | 单向 | 小 (<200W) | \(V_i\) | 需复位 | 简单 | 小功率开关电源 |
| **正激式** | 单向 | 中 (<750W) | \(V_i\) | 需复位绕组 | 简单 | 中、小功率开关电源 |
| **推挽式** | 双向 | 大 | \(2V_i\) | 有 | 简单 | 低输入电压大、中功率 |
| **半桥式** | 双向 | 大 | \(V_i\) | 无 | 复杂 (有高端) | 高输入电压大功率 |
| **全桥式** | 双向 | 最大 | \(V_i\) | 可串电容解决 | 最复杂 | 工业级大功率 |

## 典型集成开关电源控制器及应用电路

### UC3842 开关电源控制器

#### 内部结构与引脚功能
&emsp;&emsp;UC3842 是一种用于单端开关电源的集成控制器，采用 8 脚 DIP 封装，具有体积小、驱动能力强、稳压范围宽、所需外围器件少等特点。其内部电路结构如图 6.4.1 所示。

&emsp;&emsp;各引脚功能如下：
1.  **V_CC (7脚):** 电源端。正常工作电压范围 \(16 \sim 30\,\text{V}\)，启动电压 \(16\,\text{V}\)，停止电压 \(10\,\text{V}\)。
2.  **GND (5脚):** 电源地。
3.  **V_REF (8脚):** 5V 基准电压输出端。
4.  **RC (4脚):** 内部振荡器的定时电阻、定时电容接入端。
5.  **FB (2脚):** 误差放大器 \(A_1\) 的反相输入端。
6.  **COMP (1脚):** 误差放大器 \(A_1\) 的输出端。
7.  **C_S (3脚):** 电流传感端。
8.  **OUT (6脚):** 脉冲输出端，采用图腾柱式输出电路。

#### 振荡频率设定
&emsp;&emsp;将定时电阻 \(R_T\) 接于 V_REF 与 RC 端之间，定时电容 \(C_T\) 接于 RC 与 GND 端之间，振荡频率 \(f\) 为：
\[f \approx \frac{1.72}{R_T C_T} \quad (6.4.1)\]

#### 应用电路（反激式）
&emsp;&emsp;图 6.4.2 示出了以 UC3842 为控制器构成的反激式开关电源电路。

##### 工作原理分析
1.  **启动：** 交流输入经整流滤波后得到约 \(310\,\text{V}\) 直流电压，经电阻 \(R_1\) 为 UC3842 提供启动电压。
2.  **工作供电：** UC3842 启动后驱动功率开关 \(\mathrm{T}_S\) 工作，隔离变压器辅助绕组 \(N_3\) 产生的感应电压经整流滤波后为 UC3842 供电，实现自馈电。
3.  **PWM 控制：** 误差放大器 \(A_1\) 将输出电压反馈值与 2.5V 基准电压比较产生误差电压。PWM 比较器 \(A_2\) 将误差电压与电流传感信号（来自电阻 \(R_S\)）进行比较，确定输出脉冲占空比。
4.  **稳压过程：** 当输出电压升高时，反馈电压升高，误差放大器输出电压降低，PWM 比较器输出脉冲占空比减小，使输出电压回降，实现稳压。

##### 改进电路
&emsp;&emsp;为提高性能和可靠性，实用电路通常增加以下辅助电路：
1.  **输入滤波电路：** 接入共模扼流圈和压敏电阻，用于滤除共模噪声和防止输入电压过高。
2.  **稳压反馈电路：** 在反馈回路串入 TL431 稳压电路，提高输出电压稳定性。
3.  **保护电路：** 增加过压、过流保护电路，一旦出现故障使 UC3842 进入停机状态。

### SG3525A 开关电源控制器

#### 内部结构与引脚功能
&emsp;&emsp;SG3525A 是一种应用广泛的开关电源集成控制器，主要用于双端开关电源，也可用于单端开关电源。采用 16 脚 DIP 封装，其内部电路结构如图 6.4.5 所示。

&emsp;&emsp;各引脚功能如下：
1.  **V_CC (15脚):** 电源端，工作电压范围 \(8 \sim 35\,\text{V}\)。
2.  **GND (12脚):** 电源地。
3.  **V_REF (16脚):** 5.1V 基准电压输出端。
4.  **RC (4脚):** 振荡器输出端。
5.  **R_T (6脚):** 定时电阻接入端。
6.  **C_T (5脚):** 定时电容接入端。
7.  **DIS (7脚):** 定时电容放电端，可外接电阻以调节死区时间。
8.  **OUT_A (11脚):** A 路输出端。
9.  **OUT_B (14脚):** B 路输出端。两路输出交替导通，相位相差 \(180^\circ\)。
10. **SYNC (3脚):** 同步振荡信号输入端。
11. **IN_- (1脚):** 误差放大器反相输入端。
12. **IN_+ (2脚):** 误差放大器同相输入端。
13. **COMP (9脚):** 误差放大器输出端。
14. **SS (8脚):** 软启动端。
15. **SD (10脚):** 停机控制端。

#### 振荡频率设定
&emsp;&emsp;振荡频率由定时电容 \(C_T\)、定时电阻 \(R_T\) 和放电电阻 \(R_D\) 共同决定：
\[f = \frac{1}{C_T (0.7 R_T + 3 R_D)} \quad (6.4.2)\]

#### 应用电路（半桥式）
&emsp;&emsp;图 6.4.6 示出了以 SG3525A 为控制器构成的半桥式开关电源电路。
1.  **供电：** 交流输入经工频变压器降压、整流滤波后得到约 \(16\,\text{V}\) 直流电压，作为 SG3525A 的辅助电源。
2.  **PWM 控制：** 误差放大器将输出电压反馈值与基准电压比较产生误差电压，PWM 比较器将其与振荡器产生的锯齿波比较，确定输出脉冲占空比。
3.  **输出驱动：** OUT_A 和 OUT_B 两路输出通过脉冲变压器 \(\mathrm{TB}_2\) 驱动半桥式电路的功率 MOSFET 交替导通。
4.  **软启动：** 外接软启动电容 \(C_{SS}\)，使电路启动时占空比逐渐增大，避免冲击。

## 电源管理技术

### 电路的开、关机管理

#### 直流开、关机控制
&emsp;&emsp;如图 6.5.1 所示，直流开、关机控制方式通过始终加电工作的开、关机管理电路控制电子装置的电源主开关 \(S_1\)，决定主电路的加电与断电。电路连接到电力线路后，无论开机或关机，电路始终耗电（开、关机管理电路的功耗和电源 \(P_1\) 的损耗）。

#### 交流开、关机控制
&emsp;&emsp;如图 6.5.2(b) 所示，交流开、关机控制方式将电源主开关 \(S_1\) 移至电源 \(P_1\) 的交流侧，并增加一个额定功率较低的辅助电源 \(P_2\) 专门为开、关机管理电路供电。关机时大功率电源 \(P_1\) 停止工作，仅辅助电源 \(P_2\) 供电，耗电量降至很低（通常在 1W 以下）。

#### 微动开关控制（电池供电）
&emsp;&emsp;如图 6.5.3 所示，采用电池供电的电子装置通过微动开关 \(S_c\) 控制电源主开关 \(T_1\)。开机时按下 \(S_c\)，开、关机管理电路加电，驱动 \(T_1\) 导通；关机时再次按下 \(S_c\)，\(T_1\) 截止，开、关机管理电路断电。可实现关机后零功耗。

### 电源转换效率管理
&emsp;&emsp;电源转换效率管理的基本思路为：在负载较大时，开关电源以较高开关频率工作，使滤波电感或隔离变压器具有很高的能量传递效率，较小体积即可满足要求；在负载较小时，降低开关频率，以降低开关器件的功率损耗，提高电源转换效率。
