# 项目简介
设计并开发一个能够辅助学习研究电能变换与控制知识的智能体。


# 任务要求
开发的智能体应具备以下能力：

- 知识问答：能准确回答电能变换与控制领域的专业问题
- 计算指导：能指导学生完成电路参数计算，采用引导式教学
- 电路分析：能分析常见电路的工作原理和特性
- 故障诊断：能帮助学生分析和定位电路故障原因
- 设计引导：能引导学生完成简单电源电路的方案设计

# 知识覆盖范围
智能体知识库应涵盖以下模块：

1. 电力电子器件（二极管、晶闸管、MOSFET、IGBT）
2. AC-DC变换电路（整流电路）
3. DC-DC变换电路(Buck、Boost、Buck-Boost）
4. DC-AC变换电路（逆变电路）
5. AC-AC变换电路（交交变频)
6. PWM控制技术
7. 软开关技术
8. 谐振变换器
9. 无线传能技术
10. 电机驱动技术
